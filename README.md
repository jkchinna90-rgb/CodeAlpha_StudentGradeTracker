# Student Grade Tracker

## Description

Student Grade Tracker is a Java console application that helps manage and analyze student marks. The program allows users to enter marks for multiple students and calculates important statistics such as average marks, highest marks, and lowest marks.

## Features

* Input marks for multiple students
* Calculate average marks
* Find highest marks
* Find lowest marks
* Display student performance summary

## Technologies Used

* Java
* Arrays
* Scanner Class

## How to Run

1. Compile the program:
   javac StudentGradeTracker.java

2. Run the program:
   java StudentGradeTracker

## Sample Output

Enter number of students: 3

Enter marks of student 1: 80

Enter marks of student 2: 90

Enter marks of student 3: 70

----- Result -----

Average Marks = 80.0

Highest Marks = 90

Lowest Marks = 70
